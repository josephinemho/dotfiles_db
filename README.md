# dotfiles
My bash environment

### Requirements:
Script is written for a RHEL/CentOS operating system. 

* `sudo yum install git -y`
* `sudo git clone https://github.com/josephinemho/dotfiles_db.git && cd dotfiles_db && sudo chmod +x boot.sh && ./boot.sh`
